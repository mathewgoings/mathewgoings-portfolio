Campus Tour Automation

Overview: End-to-end automated HQ campus scheduling. New hires receive an invite to schedule; confirmations & ambassador notifications are automated. Reschedules are managed via a Meeting Maker Power App that writes back to SharePoint and updates calendar invites in real time. GUID tokens secure each reschedule link for compliance tracking.

Magenta Sparkle — Desk Drop Automation

Overview: Leader opt-in on employee start triggers a SharePoint writeback and weekly ambassador digests per campus. Eliminates manual coordination and provides an auditable program flow.

Mechanics & Architecture

  • Workday / Enboarder event triggers → distribution list
  • Power Automate flows to populate SharePoint lists
  • Power App for reschedule UX with GUID token mapping
  • Weekly ambassador digests via automated emails
